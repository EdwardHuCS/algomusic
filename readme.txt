Setup
1. sudo pip install -r requirements.txt
2. Create ~/.music21rc using instructions here: http://web.mit.edu/music21/doc/tutorials/environment.html
3. Add midipath to music21rc, eg: <preference name="midiPath" value="/usr/bin/vlc" />
